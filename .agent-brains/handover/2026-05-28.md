### Decisions Made

- **WiX over Velopack**: Velopack installs HKCU (per-user) which causes the app to be invisible in
  Settings > Apps on many Windows configurations. WiX MSI with `perMachine` scope writes HKLM.
- **WiX 6.0.2 over 7.0.0**: WiX 7.0.0 was released April 2026 but introduced a commercial maintenance
  fee. 6.0.2 is the widely-used stable version.
- **Browser-open stub for updates**: Full semver version checker deferred to a future release. The
  `Check for Updates` toolbar button opens the GitHub Releases page; no auto-download behaviour.
- **`WixUI_Mondo` dialog set**: Chosen because it provides both directory picker and feature tree
  in a single wizard flow (unlike `WixUI_InstallDir` which has no feature tree, or `WixUI_FeatureTree`
  which has no path picker).
